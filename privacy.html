Privacy Policy — S*A AI Assistant

Last updated: 2026-02-04

Summary

S*A AI Assistant is a voice-first personal assistant. You can enable Local‑Only mode to keep processing on-device and avoid cloud transfers.

Data We Collect

  • Audio input (voice commands) — stays on device if Local‑Only is enabled.
  • Generated content (text, images, audio) when cloud features are enabled.
  • Contacts (optional) when you grant access for calling/messaging.

User Controls

  • Toggle Local‑Only mode
  • Clear history and queued actions
  • Revoke permissions in Android settings
